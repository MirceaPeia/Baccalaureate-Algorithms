Divizibilitate. Numere Prime. Algoritmul lui Euclid

Algoritmi principlai necesari pentru rezolvarea problemelor din aceasta categorie:

Algoritmul de descompunere in factori primi:

int main()
{
int n,p,d=2;
cin>>n;	
while(n>1)
{p = 0;
while(n % d == 0)
{
p++;
n=n/d;
}
if(p)
cout <<d<<" "<<p<< endl;
d++;
if(n>1&&d*d>n)
d=n; 
}
return 0;
}

Algoritmul de verificare daca un numar este prim:

int main()                 
{                          
int n,ok=0;                 
cout<<"Introdu n"; cin>>n; 
for (int i=2; i<=n/2; i++)   
if (n%i==0)                
ok=1;                       
if (ok==1&&n>1)             
cout<<"Numarul este prim"; 
else                       
cout<<"Numarul nu este prim";
return 0;
}

Detereminare CMMDC prin scaderi repetate:

int a,b;
int main()
{cin>>a>>b;
while(a!=b)
{if(a>b)
a=a-b;
else b=b-a;
}
cout<<a;   
return 0; 
}

Determinare CMMDC folosind algoritmul lui Euclid:

int a,b,r;
int main()
{cin>>a>>b;
r=a%b;

while(r!=0) 
{a=b;
b=r;
r=a%b;}
cout<<b;
return 0;   
}

Probleme rezolvate ce se bazeaza pe secvenele de cod prezentate mai sus:

  • Subprogramul f are ca parametru pe numarul natural n. Acest subprograma returneaza 1 daca numarul n este numar perfect sau 0 daca acesta nu indeplineste aceasta proprietate. Un numar este perfect el este egal cu suma divizorilui lui.

    Rezolvare

  • Subprogramul divcomuni are ca paramentru numarul natural x. Subprogramul afiseaza pe ecran un numar natural care are aceasi divizori primi ca numarul x dat.

    Rezolvare

  • Subprogramul triplete are trei parametri, a, b și c, prin care primeste câte un numar natural din intervalul [0,104]. Subprogramul afisează pe ecran toate tripletele de numere naturale, x, y şi z, solutii ale ecuatiei x+y+z=c, cu proprietatea că a≤x≤y≤z≤b. Fiecare triplet afișat este incadrat intre acolade, iar numerele x, y și z sunt afisate în această ordine, separate prin cate o virgula. Dacă nu există niciun triplet cu proprietatea ceruta, se afisează mesajul nu exista. Scrieți definitia completa a subprogramului.
    Exemplu:daca a=2, b=4 şi c=8, se afiseaza pe ecran tripletele {2,2,4} {2,3,3} iar daca a=5, b=8, c=8 se afiseaza pe ecran mesajul: nu exista.

    Rezolvare

Probleme nerezolvate ce se bazeaza pe secvenele de cod prezentate mai sus:

  • Se citesc doua numere naturale a si b (a mai mic decat b). Afisati numerele din intervalul [a,b] care au proprietatea ca au numar maxim de divizori primi.
    Exemplu: a=30, b=45 => 30, 42 (au cate 3 divizori primi, iar restul numerelor au mai putini)

  • Se citesc doua numere naturale a si b (a mai mic decat b). Afisati numerele din intervalul [a,b] care au proprietatea ca au numar maxim de divizori.
    Exemplu: a=10, b=20 => 12, 18, 20 (au cate 6 divizori, iar restul numerelor au mai putini)

  • Sa se calculeze cmmdc al 3 numere a, b si c fara a calcula cmmdc a doua dintre ele si apoi cu al treilea.
